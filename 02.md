### 1. TouchDesigner

|  メニュー名  |  内容  |

## オペレータ一覧

### [COMP] ... Components

プログラムの下地を作るオペレータです。3D Objects/Panels/Otherの3属性に別れています。<br/>
3D Objectsは3D環境を構築するオペレータ、<br/>
Panelsはユーザインターフェースの作成、<br/>
Othersはそれ以外になります。

#### 3D Objects

| オペレータ名 | 機能 |
| ---- | ---- |
| Ambient Light | アンビエントライト。特定のシーンの環境光の色と強度を表す |
| Blend | 複数のキャラクターアニメーションのための入力をスケッチのように切り替える |
| Bone | Character Toolsの一部、3Dモデルのボーン |
| Camera | レンダリングする際のカメラ |
| Camera Blend | 複数のカメラのレンダリングをブレンド（アニメーション可） |
| Environment Light | 環境光を扱うライトを作成するコンポーネント |
| Geometry | SOPを入れる箱、TouchDesignerの中では、スタジオのイメージ |
| Handle | ボーンを操作するツール Handle Chopと一緒に機能する |
| Light | レンダリングする際のライト |
| Null | 何もしない |
| Shared Mem In | メモリー上でCOMPを共有 |
| Shared Mem Out | メモリー上でCOMPを共有 |

#### Panels

| オペレータ名 | 機能 |
| ---- | ---- |
| Button | ボタンを作成する |
| Container | インターフェースを作るためのパネルコンポーネントの基本 |
| Field | テキストを表現するパネルコンポーネント |
| List | リストを生成するUIコンポーネント |
| OP Viewer | オペレータの表示をパネル化する |
| Parameter | オペレータのパラメータ画面をコンポーネント化する |
| Select | パネルコンポーネントを参照する |
| Slider | スライダーのUIを作成する |
| Table | テーブルの機能を持つUIを作るパネル |

#### Other

| オペレータ名 | 機能 |
| ---- | ---- |
| Animation | キーフレームを打ち、シーケンスを持つ値を作成する |
| Base | パネル、3Dのパラメータのいずれも持たないただの箱 |
| Replicator | コンポーネントの分身を作る |
| Time | 各コンポーネントが独自のタイムラインを持つことを可能に |
| Window | モニターの外部出力関係を設定する |

### [CHOP] ... Channel Operators

数値として処理できるものや、デバイスからの信号を数値化して入出力するオペレータ群。
音声、Kinect、RealSenseなどのセンサーの値も処理することができる。ほかのオペレータの値をCHOPで数値化することができる。

| オペレータ名 | 機能 |
| ---- | ---- |
| Analyze | 入力信号を解析し平均化 / 最大値 / 音圧などの値に変換する |
| Angle | 角度の値をラジアン / ベクトル / クオータニオンなどに変換 |
| Attribute | 入力のアトリビュートの削除やアップデート。現在はクオータニオンのタイプのみサポート |
| Audio Band EQ | 16バンドのイコライザ |
| Audio Device In | サウンドデバイスから音に入力を得る |
| Audio Device Out | サウンドデバイスから音の入力を出力する / マルチチャンネル対応 |
| Audio Dynamics | コンプレッサ / リミッタなどオーディオダイナミクスを調整 |
| Audio File In | .mp3 .aif .aiff .wav .m4a などオーディオファイルを読み込む。URLによるweb上のオーディオファイルの読み込みも可能 |
| Audio Filter | ローパス / ハイパス などオーディオにフィルターをかける |
| Audio Movie | Movie File In TOP　で読みだした動画の音声ファイルを得る |
| Audio Ocillator | サイン波 / のこぎり波 などの音声の波形を生成する |
| Audio Para EQ | ターゲットの周波数を指定しイコライズする |
| Audio Play | 読み込んだオーディオファイルをトリガーで再生する |
| Audio Spectrum | 入力された音声を周波数分解する |
| Audio Stream In | RTSPサーバーから音源をストリーミング配信する |

| Beat | 周期的なランプ / パルス などを生成する |
| BlackTrax | BlackTrax社のモーションキャプチャシステムの値を取得 |

| Clip | Clip Blender CHOP とともに使う |

| Copy | 1入力目の値に沿って2入力目の値のコピーを生成 |
| Count | 入力信号の入力回数をカウントする |
| CPlusPlus | C++で作成したdllを読み込みオリジナルCHOP機能を生成 |
| Cross | 2つの入力チャンネルをブレンド。switchに似ているが、Crossでは入力の補間ができる |
| Cycle | 入力波形のループを作る |
| DAT to | Table形式のデータをCHOPチャンネルデータに変換する |
| Delay | 入力に対して、遅れた信号を出力する |
| Delete | 入力データからチャンネル全体 / 個々のサンプルを削除する |
| DMX In | DMX信号の入力を受け取る |
| DMX Out | DMX信号の出力を出す |
